# Game Audio Assets

Place the following files in this folder for in-game sound effects and music.

## Sound effects (Settings → Sound volume)

| File | Usage |
|------|--------|
| `correct_answer.wav` | Played when the player answers a question correctly |
| `wrong_answer.wav` | Played when the player answers incorrectly |
| `player_win.wav` | Played when the player finishes with health remaining (win) |
| `player_lose.wav` | Played when the player runs out of health or time (lose) |

## Background music (Settings → Music volume)

| File | Usage |
|------|--------|
| `Gameboy.mp3` | Looping BGM during gameplay (city map) |

Volume for **Sound** and **Music** is controlled in **Settings** (sliders). If a file is missing, that sound will be skipped without breaking the game.
